# Semantic Layout Project

Адаптивный лендинг кофейни с семантической вёрсткой.

## Цель работы
Закрепить навыки семантической вёрстки и построения адаптивного лейаута с помощью CSS Grid и Flexbox.

## Технологии
- HTML5 (семантические теги)
- CSS3 (Grid, Flexbox, медиа‑запросы)
- normalize.css для сброса стилей

## Особенности реализации

**Семантика:**
- `$<header>$` — шапка сайта с навигацией
- `$<nav>$` — навигационное меню
- `$<main>$` — основной контент
- `$<section>$` — тематические блоки
- `$<article>$` — карточки товаров
- `$<footer>$` — подвал сайта

**Лейаут:**
- CSS Grid — основная сетка страницы
- Flexbox — навигация и карточки товаров
- Медиа‑запросы — адаптивность
